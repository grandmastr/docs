## Authentication

All API endpoints are authenticated using Bearer tokens. Include your Elastic API key in the Authorization header:

```json
{
  "Authorization": "Bearer YOUR_ELASTIC_API_KEY"
}
```

You can find your API key in the Elastic Copilot extension settings or by logging into your Elastic account.
