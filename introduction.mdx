# Welcome to Elastic Copilot

Elastic Copilot is a comprehensive AI-powered software engineering assistant that transforms how you write, understand, and debug code. Built as a VSCode extension, it provides intelligent code understanding, real-time assistance, and powerful automation capabilities.

## What is Elastic Copilot?

Elastic Copilot is your intelligent coding companion that:

- **Understands your codebase** through advanced tool-based analysis
- **Provides AI-powered assistance** for explaining, fixing, and improving code
- **Offers intelligent code completion** with contextual suggestions
- **Automates repetitive tasks** through terminal integration and browser automation
- **Extends capabilities** via Model Context Protocol (MCP) integration
- **Works out of the box** - no API keys required

## How It Works

Elastic Copilot operates in two modes:

### Subscription Mode (Recommended)
- **No API keys required** - authenticate with your Elastic account
- **Pay-per-use pricing** through your subscription
- **Access to latest models** including Claude, GPT-4, and more
- **Managed infrastructure** - no setup required

### Direct Provider Mode
- **Bring your own API keys** for direct provider access
- **Full control** over model selection and usage
- **Support for multiple providers**: Anthropic, OpenAI, Google, AWS Bedrock, and more
